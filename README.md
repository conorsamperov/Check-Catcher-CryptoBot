## 🎯 Описание

**ASTYX CATCHER** - это специализированный Telegram бот, разработанный на библиотеке Pyrogram, который автоматически отслеживает и активирует чеки CryptoBot в Telegram. Бот работает в фоновом режиме, мгновенно реагируя на появление новых чеков в отслеживаемых чатах.

### Почему стоит выбрать этого бота?
- ⚡ **Мгновенная реакция** - активация чеков за доли секунды
- 🔒 **Безопасность** - полный контроль над аккаунтом
- 📊 **Статистика** - подробный отчет о всех активированных чеках

## ✨ Возможности

- ✅ Автоматическое обнаружение чеков CryptoBot в чатах
- ✅ Мгновенная активация найденных чеков
- ✅ Поддержка множества чатов одновременно
- ✅ Уведомления об успешной активации

## 🚀 Установка

### Предварительные требования
- Python 3.11 или выше
- Telegram аккаунт
- API ID и API Hash от my.telegram.org

### Пошаговая установка

1. **Клонируйте репозиторий**
```bash
git clone https://github.com/yourusername/cryptobot-cheque-hunter.git
cd cryptobot-cheque-hunter
```

2. **Создайте виртуальное окружение**
```bash
python -m venv venv
source venv/bin/activate  # Для Linux/Mac
venv\Scripts\activate     # Для Windows
```

3. **Установите зависимости**
```bash
pip install -r requirements.txt
```

4. **Настройте конфигурацию**
```bash
API_ID = Ваш ID с my.telegram.org
API_HASH = 'Ваш HASH с my.telegram.org'
OWNER_ID = Тelegram ID владельца
```

## ⚙️ Настройка

### Получение API credentials

1. Перейдите на my.telegram.org
2. Войдите в свой аккаунт
3. Перейдите в раздел "API Development"
4. Создайте новое приложение
5. Скопируйте api_id и api_hash
